# CPU Scheduling Comparison Project

## Project Description
A web-based application for comparing Priority Scheduling and SRTF algorithms.

## Features
- Priority Scheduling
- SRTF Scheduling
- Gantt Chart
- Waiting Time Calculation
- Turnaround Time Calculation
- Response Time Calculation 
- AVG WT , TAT , RT Calculation
- Input Validation

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download project
2. Open index.html in browser
